# CV Design System

## Core Design Principles
- **Style**: Minimalist Modern
- **Color Palette**: 
  - Primary: #2D3436 (Charcoal)
  - Secondary: #D4AF37 (Gold)
  - Background: #FFFFFF
  - Accent: #E8B4B8 (Soft Pink) - for highlights
- **Typography**: 
  - Headings: "Cormorant Garamond", serif
  - Body: "Inter", sans-serif
  - Code/Technical: "Fira Code", monospace
- **Spacing System**: 
  - Base unit: 8px
  - Section spacing: 32px (4 units)
  - Element spacing: 8px (1 unit), 16px (2 units), 24px (3 units)

## CV Page Structure
1. Header
   - Full Name (large, prominent)
   - Professional Title (smaller, muted)
   - Contact Information (icons with text: email, phone, location, LinkedIn, GitHub)

2. Professional Summary
   - Brief 2-3 sentence overview of expertise and value proposition

3. Key Skills
   - Categorized (Technical, Soft Skills, Languages)
   - Proficiency indicators (bars, stars, or levels)
   - Optional: progress bars or skill tags

4. Professional Experience
   - Reverse chronological order
   - Each entry: Company, Position, Dates, Location
   - Bullet points highlighting achievements (quantified where possible)

5. Education
   - Degree, Institution, Dates, Location
   - Relevant coursework or honors (optional)

6. Additional Sections (as needed)
   - Projects, Certifications, Publications, Awards, Volunteer Work, Languages

## Design Notes
- Use ample white space for readability
- Consistent alignment (left-aligned for body, centered for name in header optional)
- Use icons sparingly for visual interest (contact section, section headers)
- Ensure sufficient color contrast for accessibility (WCAG AA)
- Avoid: 
  - Neon colors
  - Excessive borders or boxes
  - More than 2 font families
  - Lengthy paragraphs (use bullet points)
  - Photos or graphics unless industry-standard (e.g., modeling, acting)
- Must include:
  - Clear visual hierarchy (name largest, then section headings, then body)
  - Mobile-responsive layout (single column on narrow screens)
  - Print-friendly CSS (optional but recommended)
